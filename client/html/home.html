Welcome to MASHYPass

A lightweight password manager built for everyday users.

Discover More
Convenient

MashyPass is quick to set up and automatically works to help you remember and improve your passwords.

Secure

MashyPass encrypts your passwords using the strongest known methods and stores them in a highly secure database.

Effective

MashyPass uses state-of-the-art methods to help you generate the strongest passwords and keep your accounts safe.

User Friendly

Streamlined and simple tools for analyzing and bolstering your password security - no expertise required!

Security is our #1 Priority

MASHYPass was developed as a team project for a Usable Security class at the University of Chicago. We performed user studies to identify security and usablitiy features important to users.
